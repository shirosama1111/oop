
8.If you create a constructor method to build a DVD by title then create a constructor method to
build a DVD by category. Does JAVA allow you to do this?
-Có Java cho phép bạn tạo 2 constructor khác nhau 1 xếp dữ liệu theo title 1 theo category
tuy nhiên việc tạo như vậy có thể gây lỗi khi lưu dữ liệu vào construct.
Java hỗ trợ nạp chồng (overloading) constructor, miễn là chúng có danh sách tham số khác nhau.
Còn nếu xây dựng 2 construct có 1 tham số title hoặc category thì ko đc.
